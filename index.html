HTML 기초

웹페이지의 기본 구조를 만드는 언어

기본 태그들:

<html> - 웹페이지 전체
<head> - 페이지 정보
<body> - 보이는 내용
<div> - 영역 나누기
<img> - 이미지 삽입
<a> - 링크 만들기

예시 코드:

<div>
  <h1>제목</h1>
  <p>내용</p>
  <a href="링크">버튼</a>
</div>
